ALPAKOWY BAZAREK – URUCHOMIENIE MONGODB (DOCKER)

1. Uruchomienie kontenera MongoDB:
docker run --name alpaca-mongo -p 27017:27017 -d mongo:6.0

2. Sprawdzenie czy kontener działa:
docker ps

3. Połączenie z bazą danych przez MongoDB Compass:
URI: mongodb://localhost:27017
nazwa: Alpakowy bazarek

4. Utworzenie bazy danych:
nazwa: alpacaApp

5. Dane są przechowywane w kolekcjach:
- alpacas
- orders
- users

6. Zatrzymanie kontenera:
docker stop alpaca-mongo

7. Ponowne uruchomienie kontenera:
docker start alpaca-mongo

Aplikacja Node.js łączy się z MongoDB przez adres:
mongodb://localhost:27017
